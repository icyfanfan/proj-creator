# 基本目录结构生成cli工具

基于webpack4 vue的项目基本目录结构生成cli工具

全局安装

```
➜  npm install -g proj-creator
```

使用

```
proj-creator simple <app_name>

将在当前目录/<app_name>下生成基础目录结构

```

或者 自定义配置
```shell
proj-creator custom <app_name>
```
